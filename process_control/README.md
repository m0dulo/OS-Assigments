# Process Control
Simulate operating system kernel management of processes and simple memory management with Best Fit algorithm.

## Usage
```bash
mkdir log
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=release
make
../bin/process -h
```
